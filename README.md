# dotfiles

## 1. Install chezmoi

[chezmoi \- chezmoi](https://www.chezmoi.io/)

```sh-session
brew install chezmoi
```

## 2. Init dotfiles

```sh-session
chezmoi init --apply --ssh hayajo
```

Please refer to [Daily operations \- chezmoi](https://www.chezmoi.io/user-guide/daily-operations/) for further operations.
